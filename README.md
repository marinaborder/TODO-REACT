TODO-REACT
Este é um projeto de estudo em react simples de lista de tarefas (To-Do List). Ele permite aos usuários adicionar, visualizar, marcar como concluídas e excluir tarefas.

Tecnologias Utilizadas
As seguintes tecnologias foram utilizadas no desenvolvimento deste projeto:

React (Criado com Create React App): Biblioteca JavaScript para construir interfaces de usuário.

JavaScript (ES6+): Linguagem de programação principal.

HTML5: Estrutura da aplicação.

CSS3: Estilização da aplicação.

Instruções para Rodar o Projeto Localmente
Siga os passos abaixo para configurar e executar o projeto em sua máquina local.

Pré-requisitos
Você deve ter o Node.js e o npm (ou Yarn) instalados em seu sistema.

1. Clonar o Repositório
Abra o terminal e execute o seguinte comando para clonar o projeto:

Bash

git clone https://github.com/marinaborder/TODO-REACT.git
2. Navegar para o Diretório do Projeto
Entre na pasta do projeto clonado:

Bash

cd TODO-REACT

3. Instalar as Dependências
Execute o comando de instalação de dependências:
Bash
npm install
